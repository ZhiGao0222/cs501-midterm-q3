# CS501 Q3 - LazyColumn Student List

This app displays a scrollable list of student names using Jetpack Compose.

## Features
- Displays a list of 20 student names
- Uses LazyColumn to create a scrollable list
- Uses the items() DSL to display list data
- Each item is shown inside a Card

## Technologies Used
- Kotlin
- Jetpack Compose
- LazyColumn
- items()

## How it works
The app defines a list of student names and displays them using a `LazyColumn`.  
The `items()` function is used to iterate through the list and create UI elements for each name.  

Each student name is displayed inside a Card, and the list can be scrolled vertically.
